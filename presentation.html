System Overview

Library OS is a comprehensive management system engineered for educational institutions, libraries, and book clubs to streamline inventory tracking, loans history, and user directory permissions.

Role-Based Access

Separates Admin controls, Librarian processing, and Member browsing.

Architecture & Stack

Vite & React Frontend

Single Page React client utilizing TailwindCSS for micro-animations and modular component layouts. Serves e-book reads and profile actions.

PORT: 80 / dev 5173

Spring Boot API

RESTful architecture utilizing Hibernate JPA for database persistence, JWT tokens for auth verification, and dynamic JSON controller mappings.

PORT: 8080 (Java 17)

MySQL Database

Relational backend schemas tracking primary tables (users, books, loans) with dynamic column upgrades via JPA ddl-auto properties.

PORT: 3306 (MySQL 8.0)

UI Upgrade: Cover Image System

We replaced plain text cards with a multi-layered cover resolution pipeline.

  • Custom Links: Option to input custom cover URLs.
  • Open Library API: Fallback to search covers by ISBN.
  • 3D Fallback Gradients: Programmatically generated cover designs styled dynamically according to the book's genre if offline or missing.
